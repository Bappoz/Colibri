## O render como sistema

```rust
fn render_system(world: &World, fb: &mut Framebuffer) {
    let (cam_t, cam) = world.single::<(&Transform, &Camera)>();
    let view = view_from(cam_t);
    let proj = perspective(cam, fb.aspect());
    for (t, mesh) in world.query::<(&Transform, &MeshHandle)>() {
        let world_m = model_matrix(t);
        raster_mesh(fb, proj * view * world_m, meshes.get(mesh.0));
    }
}
```

E o `spin_system` substitui a linha `self.theta += dt`:

```rust
fn spin_system(world: &mut World, dt: f32) {
    for (t, spin) in world.query::<(&mut Transform, &Spin)>() {
        t.rotation.set_y(t.rotation.y() + spin.0 * dt);
    }
}
```

> **Onde o `Engine` fica?** Ele encolhe para um **host**: dono da janela, do
> framebuffer e do `World`. No `about_to_wait`, chama `schedule.run(world, dt)` e
> `render_system`. `App` continua sГі roteando eventos do winit.
